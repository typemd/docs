title: 工具指令
description: 初始化、統計、設定等工具指令。
---

初始化 vault、檢視統計、管理設定和啟動 web 伺服器。

| 指令 | 說明 |
|------|------|
| [`tmd init`](/zh-tw/cli/init/) | 初始化新的 vault |
| [`tmd stats`](/zh-tw/cli/stats/) | Vault 整體摘要與指定 type 的屬性彙總統計 |
| [`tmd config`](/zh-tw/cli/config/) | 從 CLI 管理 vault 設定 |
| [`tmd instructions`](/zh-tw/cli/instructions/) | 輸出嵌入的 skill 指令並附帶 vault context |
| [`tmd log`](/zh-tw/cli/log/) | 顯示特定 Object 的 git 提交歷史 |
| [`tmd serve`](/zh-tw/cli/serve/) | 啟動 Web UI 伺服器 |
