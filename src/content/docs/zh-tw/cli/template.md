title: Template 指令
description: 管理 object template 的指令。
---

管理 object template，提供可重複使用的 frontmatter 和正文內容。

| 指令 | 說明 |
|------|------|
| [`tmd template list`](/zh-tw/cli/template-list/) | 列出可用的 template |
| [`tmd template show`](/zh-tw/cli/template-show/) | 顯示 template 的內容 |
| [`tmd template create`](/zh-tw/cli/template-create/) | 建立新的 template 檔案 |
| [`tmd template delete`](/zh-tw/cli/template-delete/) | 刪除 template 檔案 |
