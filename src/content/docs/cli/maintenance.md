description: Commands for migrating, formatting, fixing, and diagnosing vault issues.

Migrate schemas, normalize formatting, fix issues, and run health checks.

| Command | Description |
|---------|-------------|
| [`tmd migrate`](/cli/migrate/) | Migrate type schemas and objects |
| [`tmd format`](/cli/format/) | Normalize object frontmatter and schema YAML formatting |
| [`tmd fix`](/cli/fix/) | Fix common issues in your vault |
| [`tmd doctor`](/cli/doctor/) | Comprehensive vault health check with diagnostics and auto-fix |
