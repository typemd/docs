---
title: Object Commands
description: Commands for creating, viewing, and listing objects.
---

Create, view, list, and search objects in your vault.

| Command | Description |
|---------|-------------|
| [`tmd object create`](/cli/create/) | Create a new object from a type schema |
| [`tmd object show`](/cli/show/) | Display an object's full information |
| [`tmd object list`](/cli/object-list/) | List all objects in the vault |
| [`tmd search`](/cli/search/) | Full-text search across your vault |
| `tmd object lock <id>` | Lock an object to prevent editing |
| `tmd object unlock <id>` | Unlock a locked object |
| `tmd object archive <id>` | Archive an object (soft delete) |
| `tmd object unarchive <id>` | Unarchive an object |
