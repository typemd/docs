title: Type Commands
description: Commands for managing type schemas.
---

Inspect, list, and validate type schemas.

| Command | Description |
|---------|-------------|
| [`tmd type show`](/cli/type-show/) | Show details of a type schema |
| [`tmd type list`](/cli/type-list/) | List all available types |
| [`tmd type validate`](/cli/validate/) | Validate type schemas, objects, relations, and wiki-links |
