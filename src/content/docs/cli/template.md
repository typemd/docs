---
title: Template Commands
description: Commands for managing object templates.
---

Manage object templates for reusable frontmatter and body content.

| Command | Description |
|---------|-------------|
| [`tmd template list`](/cli/template-list/) | List available templates |
| [`tmd template show`](/cli/template-show/) | Display a template's content |
| [`tmd template create`](/cli/template-create/) | Create a new template file |
| [`tmd template delete`](/cli/template-delete/) | Delete a template file |
