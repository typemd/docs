Lists all objects in the vault.

```bash
tmd object list
```

Example output:

```
book/golang-in-action-01jqr3k5mpbvn8e0f2g7h9txyz
book/clean-code-01jqr4a2bcdef0123456789xyz
person/alan-donovan-01jqr3k5mpbvn8e0f2g7h9txyz
person/robert-martin-01jqr4b3cdefg1234567890abc
```

## Flags

| Flag | Description |
|------|-------------|
| `--json` | Output the list in JSON format |
